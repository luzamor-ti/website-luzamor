# Quickstart

O guia de início rápido agora faz parte da documentação principal em [docs/index.html](docs/index.html).

Se você quer subir o projeto agora mesmo:

1. Instale dependências com `npm install`.
2. Configure `.env.local`.
3. Execute `npm run dev`.

Confira a visão completa em [docs/index.html](docs/index.html).
